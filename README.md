# Tic Tac Toe Web Game

## Overview
This project is a simple Tic Tac Toe game built with **HTML, CSS, and JavaScript**.  
Users can play X and O on the same board. The game detects wins, losses, and draws,  
and allows the user to reset the game at any time.

---

## Tech Stack
- HTML  
- CSS  
- JavaScript  

---

## Features
- Two-player Tic Tac Toe  
- Responsive and animated UI  
- Real-time turn switching  
- Win detection  
- Draw detection  
- Reset button to restart the game  

---

## How to Run (Step-by-Step)
1. Download or clone the repository  
2. Open the project folder  
3. Double-click **`index.html`**  
4. The game will open in your default web browser  
5. Click on any cell to place **X** or **O**  
6. The program will automatically check for a win or draw  
7. Press **Reset Game** to start a new round  

---

## How It Works (Step-by-Step)
1. Each board cell is linked to an index in a 9-element array  
2. When the user clicks a cell, the JavaScript updates the array  
3. The script checks all 8 winning conditions  
4. If a winning condition matches, the game displays the winner  
5. If no empty spaces remain, the game declares a draw  
6. Reset button clears the board and resets game state
